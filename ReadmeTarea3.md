El proyecto consiste en la implementación de un Árbol Binario de Búsqueda (ABB) utilizando el lenguaje Python. Su objetivo es permitir la creación, manipulación y visualización de esta estructura de datos mediante una interfaz de línea de comandos.

El programa ofrece funcionalidades básicas como la inserción, búsqueda y eliminación de elementos dentro del árbol, respetando las propiedades propias de un ABB. Asimismo, permite la carga de datos desde un archivo en formato txt, facilitando la construcción del árbol de manera automatizada.

Adicionalmente, el sistema integra la herramienta Graphviz para generar una representación gráfica del árbol, permitiendo visualizar de forma clara la estructura jerárquica y la relación entre los nodos. Durante la búsqueda, el nodo encontrado se resalta para mejorar la interpretación visual..
